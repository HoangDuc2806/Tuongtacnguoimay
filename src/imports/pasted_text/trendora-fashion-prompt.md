MASTER UI/UX DESIGN PROMPT – EcoWear FASHION E-COMMERCE
Project Title

EcoWear Fashion – Modern E-commerce Product Catalogue & Shopping Platform

Project Overview

Design a complete, responsive, modern, and visually appealing fashion e-commerce web application inspired by the layout structure and clean aesthetic of a premium online shopping platform similar to EcoWear.

The application should support:

product discovery
browsing by category
advanced filtering
sorting
wishlist
cart
checkout
order tracking
user profile management
seller/admin management
responsive desktop/mobile experiences

This project should feel like a real-world fashion retail product for young adults and university students.

The design must emphasise:

clean modern UI
strong UX hierarchy
responsive layouts
component consistency
real-world e-commerce interactions
micro-interaction ready states
design system scalability
