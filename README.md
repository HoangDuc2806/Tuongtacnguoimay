

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
  # Quá trình thực hiện giao diện Wishlist và Cart

- Sử dụng công cụ **CodeTea** để generate giao diện ban đầu cho 2 trang **Wishlist** và **Cart**.
- Sau khi generate, mã nguồn được đưa vào các file giao diện trong dự án **React**.
- Tiến hành chỉnh sửa lại bố cục, CSS và component để phù hợp với cấu trúc project.
- Kết nối điều hướng giữa các trang và hoàn thiện giao diện hiển thị.

## Các trang đã thực hiện

- Wishlist
- Cart

## Kết luận

Sử dụng CodeTea để hỗ trợ tạo giao diện nhanh, sau đó chỉnh sửa và tích hợp thủ công vào dự án React để đảm bảo giao diện hoạt động đúng yêu cầu.
